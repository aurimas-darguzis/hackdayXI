This component creates a button. You can specify various properties of it. By default, the background colour will be transparent. You can add primary or success as type property. An icon is optional, but if you do want it - just pass the class name as icon property to the component. You can also specify the size - just specify large, medium, small or xsmall. This button emits an event to the parent component, so the functionality should be implemented by the parent class.
